# Study-Tracker v1.2
Python-based study tracking application featuring session timers, topic logging, daily goal tracking, motivational quotes, and progress monitoring to help students build consistent study habits.
Built with a focus on clarity, modularity, and motivation, the app transforms daily learning into a structured, goal‑oriented experience.
💡 Key Features
Session Timer — Set custom study durations for each topic with real‑time countdowns and progress indicators.

Topic Logging — Record each study topic, duration, and completion status automatically.

Daily Goals — Track total study time and compare against daily targets.

Summary Reports — Generate end‑of‑day summaries showing total time spent and topics covered.

Sound Alerts — Built‑in audio notifications when timers end or new prompts appear (using afplay on macOS and playsound for Windows).

Study Tips — Motivational insights and reminders to maintain consistency.

Persistent Logs — Automatically saves all sessions for future review.
